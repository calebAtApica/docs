# apicactl

The Apica Ascent command line toolkit, apicactl, allows you to run commands against Apica Ascent Observability stack.

You can&#x20;

* Real-time streaming of logs
* Query historical application logs&#x20;
* Search your log data.
* View Events
* Manage Dashboards
* Create event rules
* Manage license
